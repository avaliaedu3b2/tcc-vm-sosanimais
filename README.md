# 🐾 SOS Animais Matão  
### Sistema Web Para Denúncia de Maus-Tratos a Animais.  

---

## 📌 Objetivo  

Criar uma aplicação web funcional para o registro e acompanhamento de denúncias de maus-tratos contra animais no município de Matão. A plataforma permite que os usuários:

- Realizem cadastro e autenticação de contas  
- Façam denúncias de forma simples e rápida  
- Informem a localização exata do ocorrido  
- Anexem fotos como evidência   
- Contribuam para a proteção e bem-estar animal  

---

## ⚙️ Funcionalidades  

✔ Cadastro e login de usuários  
✔ Registro de denúncias de maus-tratos  
✔ Envio de localização geográfica  
✔ Upload de fotos como evidência   
✔ Interface simples, intuitiva e responsiva  
✔ Atualização dos dados em tempo real  

---

## 💻 Tecnologias Utilizadas  

| Tecnologia | Descrição |
|-----------|----------|
| Python | Linguagem utilizada para desenvolvimento da lógica e processamento |
| Firebase | Plataforma para backend e integração de serviços |
| Firestore Database | Banco de dados em tempo real |
| Google Cloud (GCP) | Infraestrutura em nuvem e deploy |

---

## 🚀 Contribuição  

Este projeto tem como objetivo incentivar a participação da comunidade no combate aos maus-tratos contra animais, promovendo mais segurança e bem-estar para os animais da cidade.
